# Grand Collective Squarespace Build Guide

This guide turns the current custom site into a Squarespace-friendly structure you can edit directly after publishing.

## Recommended Squarespace Setup

Use a service business, portfolio, or salon-style template. You do not need ecommerce unless you plan to sell products.

Create these pages:
- Home
- Abbie Brown
- Mariah Brasel
- Mary Elizabeth Brooks
- Reshae Simmons
- Contact / Book

Navigation:
- Home
- Team
- Gallery
- Contact
- Book Now

The Team link can scroll to the Team section on the Home page.

## Squarespace Editing Tips

- Use page sections for each stylist instead of trying to create a complex CMS.
- Use buttons for booking and social links.
- Use accordion blocks if service lists feel too long.
- Use gallery blocks for portfolios.
- Keep stylist pages consistent so the site feels polished.
- After publishing, you can edit directly in Squarespace and click Save/Publish without GitHub or code.
